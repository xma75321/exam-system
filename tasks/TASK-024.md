## 详细要求

### frontend/src/components/SubmitConfirmModal.tsx
- 显示答题统计：
  - 总题数
  - 已答数
  - 未答数
- 未答题警告（如有未答题）
- 二次确认按钮
- 取消按钮
- 提交中显示加载状态

### frontend/src/components/ExamComplete.tsx
- 提交成功后显示
- 大号 "提交成功" 提示
- 考试信息摘要
- "查看结果" 按钮（跳转到结果页）
- "返回考试列表" 按钮

### frontend/src/app/exams/[id]/take/page.tsx（修改）
- 提交流程：
  1. 点击"提交考试" → 弹出确认弹窗
  2. 确认后 → 先调用 saveNow() 保存最新答案
  3. 调用提交 API
  4. 成功 → 停止计时器、自动保存 → 显示完成页面
  5. 失败 → Toast 错误提示
- 时间到期自动提交流程：
  1. 计时器到期触发
  2. 强制保存答案
  3. 调用提交 API
  4. 显示完成页面

## 验收标准
1. 点击提交按钮弹出确认弹窗
2. 弹窗正确显示已答/未答统计
3. 有未答题时显示警告
4. 确认后成功提交
5. 提交后显示完成页面
6. 时间到期自动提交
7. 提交后无法再修改答案

## 测试方法
```bash

# 4. 确认提交
# 5. 验证完成页面显示
# 6. 点击"查看结果"跳转
```
